The replication folder contains all the "publicly-available" code and data files that were used for obtaining the results reported Tables 1-8.

Names of all executable files start with "Main_".

The data on US exports and imports are compiled by Schott (2008) and can be downloaded from "https://www.dropbox.com/sh/jj55pbx39eq8bx9/AACrcB7cc7KdRFJYurkZItgHa?dl=0"

The BACI data can be downloaded from "http://www.cepii.fr/CEPII/en/bdd_modele/presentation.asp?id=37"

The data on industry-country flows of goods and services  (used in Section 6) can be accessed through http://www.wiod.org/protected3/data/wiot_analytic/wiot08_row_apr12.xlsx

The data on Rauch's classification is taken from "https://econweb.ucsd.edu/~jrauch/rauchclass/Rauch_classification_revised.csv"

The data on gravity variables can be downloaded from "http://www.cepii.fr/CEPII/en/bdd_modele/presentation.asp?id=8"

The BEC product classification can be accessed at "https://unstats.un.org/unsd/tradekb/Knowledgebase/50090/Intermediate-Goods-in-Trade-Statistics"

The data on the price of non-traded services from the United Nation's International Comparison Program (ICP) can be downloaded from "https://www.worldbank.org/en/programs/icp#5"
